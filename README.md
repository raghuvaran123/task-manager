# ğŸ“Œ Team & Task Management API (NestJS)

This is a backend API built using **NestJS** and **TypeScript** to manage:

- Teams  
- Users (members)  
- Tasks  
- Task Assignees  

A task does **not** belong to a team directly.  
Instead, each task has **assignees** (users), and users can belong to teams.

- Get tasks, teams, users
